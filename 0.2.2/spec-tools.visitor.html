visit

multimethod

Walk a spec definition. Takes two arguments, the spec and the accept function, and returns the result of calling the accept function.

The accept function is called with three arguments: the dispatch term for the spec (see below), the spec itself and the vector with the results of recursively walking the children of the spec.

The dispatch term is one of the following * if the spec is a function call: a fully qualified symbol for the function with the following exceptions: - cljs.core symbols are converted to clojure.core symbols - cljs.spec.alpha symbols are converted to clojure.spec symbols * if the spec is a set: :spec-tools.visitor/set * otherwise: the spec itself
