swagger-spec

(swagger-spec x)(swagger-spec x options)

Transforms data into a swagger2 spec. Input data must conform to the Swagger2 Spec (http://swagger.io/specification/) with a exception that it can have any qualified keywords that are expanded with the spec-tools.swagger.core/expand multimethod.

transform

(transform spec)(transform spec options)

Generate Swagger schema matching the given clojure.spec spec.

Since clojure.spec is more expressive than Swagger schemas, everything that satisfies the spec should satisfy the resulting schema, but the converse is not true.
